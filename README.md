# node（express）+ mongo

运行项目：
<br/>
1. 首先你需要安装mongodb,并且配置环境变量
<br/>
2. cmd中启动mongo： mongod --dbpath "c:\mongo"(目录自己创建一个)
<br/>
3.安装依赖：yarn（npm || cnpm） install 
<br/>
4.运行：node app.js
<br/>
5.启动http://localhost:3000/
<br/>
6.默认你的本地mongo是没有数据的，你可以在前台自己注册几个(包括留言),会自动存入mongo
<br/>
7.推荐用 robo-3t 看数据库数据
